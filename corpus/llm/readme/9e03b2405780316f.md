# Lumen: A Static Site Generator for Photography Portfolios

Lumen is a powerful yet lightweight static site generator designed specifically for photographers and their portfolios. It's built with simplicity and efficiency in mind, making it easy to manage your photography projects and present them beautifully online.

## Getting Started

### Installation

To get started with Lumen, you'll first need to install Ruby and the necessary dependencies. You can then install Lumen as a Ruby gem:

```sh
gem install lumen
```

### Project Structure

Lumen expects a specific folder structure for your projects. Here's an example of how your project directory might look:

```
/my-photography-portfolio
    ├── _config.yml  # Configuration file
    ├── index.md     # Frontpage
    ├── gallery1
        ├── images  # All images in this gallery go here
        │   └── img01.jpg
        │   └── img02.jpg
        └── index.md  # Gallery description or page
    ├── gallery2
        ├── images  # Another set of images for a different gallery
        │   └── img03.jpg
        │   └── img04.jpg
        └── index.md  # Gallery description or page
```

### Configuration

Lumen's primary configuration is stored in `_config.yml`. This file allows you to set key parameters for your portfolio, such as the site title, theme, and output directory.

Example `_config.yml`:

```yaml
title: "My Photography Portfolio"
theme: "default"
output_dir: "_site"
```

### Build Command

To build your portfolio, simply run the following command from within your project directory:

```sh
lumen build
```

This will process all your galleries and generate static HTML files in the specified output directory.

### Local Preview Server

For local development, you can start a simple web server to preview your site. Run the following command:

```sh
lumen serve
```

This will start a local HTTP server on `http://localhost:4000`. You can access your portfolio locally by navigating to this URL in any web browser.

### Thumbnail Generation

Lumen automatically generates thumbnails for each image in your galleries. These thumbnails are stored alongside the original images and are used to display smaller versions of your photos in galleries or on pages.

### EXIF-Based Caption Extraction

Lumen supports extracting captions from EXIF metadata embedded within your images. This allows you to include descriptive information directly in your images, which can then be extracted automatically during the build process.

To enable this feature, ensure that each image has a valid caption field in its EXIF data. Lumen will use this caption for displaying image descriptions on your portfolio pages.

### Customization

You can customize various aspects of your portfolio using the `_config.yml` file:

- **Title**: Set the title of your portfolio.
- **Theme**: Choose from available themes or create a custom one.
- **Output Directory**: Specify where to place the generated static files.

## Usage Example

Here's an example `index.md` and `gallery1/index.md` for your portfolio:

```markdown
# Welcome to My Photography Portfolio!

A brief introduction about yourself and what this portfolio is all about.
```

For a gallery page, you might have something like this:

```markdown
---
title: "Nature Photography"
---

## Introduction

This section can include a short description of the gallery or its theme.

### Images

![My Image Caption](images/img01.jpg)

![Another Image Caption](images/img02.jpg)
```

Each image file should be named appropriately, and Lumen will handle the rest to ensure your portfolio is structured beautifully.

## Conclusion

Lumen streamlines the process of building a photography portfolio by focusing on simplicity and efficiency. With its intuitive folder structure and powerful configuration options, you can easily manage and showcase your work without sacrificing quality or functionality.

Happy shooting and building!
