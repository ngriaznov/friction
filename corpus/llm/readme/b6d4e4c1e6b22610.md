# wavtag: A Lightweight C Library for WAV Metadata Handling

## Overview

`wavtag` is a small, efficient C library designed to read and write metadata chunks (commonly referred to as LIST/INFO tags) in WAV audio files without altering the actual audio samples. This makes it ideal for applications requiring modification of metadata fields such as artist, title, or comment in WAV files.

## Installation

### Prerequisites
- A C compiler (e.g., `gcc`, `clang`)
- Make utility (`make`)

### Building and Installing

1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/wavtag.git
   cd wavtag
   ```

2. Build the library using `make`:
   ```sh
   make
   ```

3. Install the built files (optional):
   ```sh
   sudo make install
   ```
   
## Public API

### Opening a WAV File for Metadata Editing

```c
#include "wavtag.h"

struct WavTag *wavtag_open(const char *filename) {
    // Opens a WAV file and returns a handle to it.
}
```

### Reading Tags from an Opened WAV File

```c
int wavtag_read(struct WavTag *handle, struct TagInfo *info) {
    // Reads metadata chunks into the `info` structure.
}
```

### Writing Modified Tags Back to the WAV File

```c
void wavtag_write(struct WavTag *handle, const struct TagInfo *info) {
    // Writes modified metadata back to the file.
}

void wavtag_close(struct WavTag *handle) {
    // Closes the handle and releases resources.
}
```

### Example Usage

```c
#include <stdio.h>
#include "wavtag.h"

int main(int argc, char **argv) {
    if (argc != 3) {
        fprintf(stderr, "Usage: %s filename new_artist\n", argv[0]);
        return 1;
    }

    struct WavTag *handle = wavtag_open(argv[1]);
    if (!handle) {
        fprintf(stderr, "Failed to open file '%s'\n", argv[1]);
        return 1;
    }

    struct TagInfo info;
    memset(&info, 0, sizeof(info));
    
    // Read existing tag
    int ret = wavtag_read(handle, &info);
    if (ret != 0) {
        fprintf(stderr, "Failed to read tags\n");
        goto cleanup;
    }

    // Modify the artist field
    strcpy(info.artist, argv[2]);

    // Write back modified tags
    wavtag_write(handle, &info);

cleanup:
    wavtag_close(handle);
    return 0;
}
```

## Byte-Order Handling

`wavtag` is designed to handle byte order differences across platforms. It ensures that metadata fields are stored in a platform-independent manner, making the library portable between different operating systems and architectures.
