## tasq: Your Plain-Text Todo Manager 

Tired of bloated todo apps that lock your tasks in proprietary formats? Want something simple, readable, and easy to track in version control? Look no further than **tasq**.

**What is tasq?**

Tasq is a lightweight command-line tool for managing your to-dos. It stores your tasks as plain text lines in a file, making them completely human-readable, diffable, and easily shareable. No cryptic databases or complicated formats here! 

**Getting Started:**

1. **Install:**
   *  Use your package manager: `pip install tasq` (Python)

2. **Locate Your Task File:**
   By default, tasq stores tasks in a file called `.tasq` in your home directory (`~/.tasq`). 

3. **Customize the Location:**
   Need to store your tasks elsewhere? Just set the `TASQ_FILE` environment variable to your preferred location before running tasq.

**Basic Commands:**

* **Add a Task:**  `tasq add "Grocery shopping - milk, eggs, bread"`
* **Complete a Task:** `tasq complete 12` (replace '12' with the task number)
* **List All Tasks:** `tasq list` 

**Powerful Features:**

* **Tagging:** Organize tasks with tags using the `@` prefix. For example: `tasq add "@work Write report" @personal "Call mom"`
* **Due Dates:** Set due dates for your tasks in YYYY-MM-DD format. Example: `tasq add "Pay bills" 2024-03-15`

**Filtering Your List:**

* **By Tag:**  `tasq list @work` (lists all tasks tagged with "work")
* **By Due Date:** `tasq list 2024-03-15` (lists tasks due on March 15, 2024)


**Why tasq?**

* **Simplicity:** Straightforward commands and a clear output make tasq easy to learn and use.
* **Readability:** Plain text files are human-readable and version-control friendly. You can easily review changes or share your task list with others. 
* **Flexibility:**  Customize the location of your task file and use tags and due dates to keep your tasks organized.


Let tasq help you conquer your to-do list with its elegant simplicity and focus on clarity!
