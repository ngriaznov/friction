Lumen
================

A Static Site Generator Built Specifically for Photography Portfolios

Lumen is a Ruby gem designed to simplify the process of creating stunning photography portfolios with minimal effort. This README will guide you through setting up and customizing Lumen to suit your needs.

Folder Convention
-----------------

To ensure seamless workflow, Lumen expects a specific folder convention:

*   `images/`: A directory for each gallery, containing all images related to that specific theme or series.
*   `.markdown/`: A directory for each page in the site, containing Markdown files with front-matter-free content.

For example, your site structure might look like this:
```plain
project/
├── images/
│   ├── gallery1/
│   │   └── image1.jpg
│   └── gallery2/
│       └── image2.jpg
└── .markdown/
    ├── index.md
    └── about-us.md
```

Build Command
--------------

To build your site, run the following command in the project root directory:
```bash
lumen build --output ./output
```
This will generate a `output` directory containing the compiled HTML files for each page.

Local Preview Server
-------------------

For easy local previewing and testing, Lumen includes a simple server that serves your site at `http://localhost:8080`.

To start the server, run:
```bash
lumen serve --port 8080
```
Thumbnail Generation
--------------------

Lumen uses ImageMagick to generate thumbnails for each image. This feature can be enabled by setting the `thumbnail` option in your configuration file (see below).

EXIF-based Caption Extraction
-----------------------------

Lumen supports EXIF-based caption extraction, allowing you to easily extract image captions from the EXIF metadata.

Configuration File
------------------

To configure Lumen to your liking, create a YAML file named `.lumen.yml` at the project root. This file contains configuration settings for the site title, theme, and output directory.

Here's an example configuration:
```yml
title: "Your Photography Portfolio"
theme: "dark"
output_dir: "./output"
thumbnail: true
```
You can customize this file to suit your needs.

Output Directory
-----------------

Lumen outputs the compiled HTML files in a directory specified by the `output_dir` option. The default output directory is set to `./output`.

Theme Options
--------------

Lumen supports various themes, each with its unique design and layout options. To use a custom theme, simply specify it in your configuration file:
```yml
theme: "your-theme-name"
```
Some popular themes are:

*   `default`: A clean and simple default theme.
*   `dark`: A dark-themed version of the default theme.

Custom Themes
-------------

To create a custom theme for Lumen, you'll need to create a new directory containing your design files. The theme should include the following directories:

*   `layouts/`: A directory containing HTML layouts for each page type (e.g., `index.html`).
*   `components/`: A directory containing reusable HTML components.
*   `assets/`: A directory containing CSS, JavaScript, and image assets.

For more information on creating custom themes, please refer to the Lumen documentation.

Conclusion
----------

Lumen is a powerful static site generator designed specifically for photography portfolios. With its simple folder convention, easy build command, local preview server, thumbnail generation, EXIF-based caption extraction, and flexible configuration options, Lumen makes it easy to create stunning online portfolios with minimal effort.
