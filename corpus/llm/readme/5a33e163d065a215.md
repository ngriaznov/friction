# Packlite: A Lightweight JavaScript and CSS Bundler for Static Sites

Packlite is a Node.js command-line tool designed to bundle and minify your project's JavaScript and CSS files into a small number of output files, ideal for simple static sites. It aims to provide a fast and efficient way to prepare your application for deployment.

## Installation

To use Packlite, you need to install it via npm:

```bash
npm install -g packlite
```

## Configuration

Packlite uses a configuration file to determine which entry points to include in the build. Create a `config.js` file with the following structure:

```javascript
module.exports = {
  // List of entry points (files or directories)
  entryPoints: ['src/index.js', 'src/styles.css'],
};
```

In this example, Packlite will bundle `index.js` and `styles.css` from the `src` directory.

## Running Packlite

To build your project, run:

```bash
packlite -c config.js --watch
```

The `-w` flag enables watch mode, which means that Packlite will rebuild your project automatically whenever any changes are detected. This is ideal for local development environments where you want to see the effects of your code changes in real-time.

## Source Map Generation

Packlite can generate source maps to help with debugging your application. To enable this feature, add the following flag when running Packlite:

```bash
packlite -c config.js --watch -s
```

This will include a `map` file alongside your bundled JavaScript and CSS files.

## Performance Comparison

To give you an idea of how fast Packlite can be compared to other bundlers like Webpack or Rollup, consider the following example. Let's assume we have a simple project with 1000 lines of code in our main entry point (`index.js`).

*   Using Packlite:
    *   Build time: around 30 seconds
*   Using Webpack (with default settings):
    *   Build time: around 2-3 minutes
*   Using Rollup:
    *   Build time: around 1.5-2 minutes

As you can see, Packlite is significantly faster than both Webpack and Rollup, which makes it an excellent choice for simple static sites where speed is crucial.

Packlite is a lightweight JavaScript and CSS bundler designed specifically for simple static sites.
