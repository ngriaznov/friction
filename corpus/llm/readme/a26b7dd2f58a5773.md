# tasq: A Simple Command-Line Todo Manager

**tasq** is a lightweight, command-line based todo manager designed to keep your tasks readable and easily manageable using version control. It stores tasks as plain lines of text in a file, making it ideal for developers who prefer simplicity and transparency.

## Getting Started

### Installation

To install tasq, you can use pip:

```bash
pip install tasq
```

### Default Task File Location

By default, tasq looks for your task file at `~/.tasq.txt`. You can change this location by setting the environment variable `TASQ_FILE` to the path of your preferred file.

## Adding Tasks

To add a new task, use the `add` command followed by the task description:

```bash
tasq add "Write documentation for API"
```

You can also tag tasks or set due dates directly in the input:

```bash
tasq add "Submit proposal @work due 2023-12-01"
```

## Completing Tasks

To mark a task as completed, use the `complete` command followed by the task index (starting from 1):

```bash
tasq complete 1
```

This will prepend an asterisk (`*`) to the task line, indicating it's done.

## Listing Tasks

To view all tasks, simply run:

```bash
tasq list
```

### Filtering by Tag or Due Date

You can filter tasks using tags or due dates. For example, to see only tasks with a specific tag `@work`:

```bash
tasq list @work
```

Or to list tasks due before a certain date:

```bash
tasq list due < 2023-12-01
```

## Tag Syntax

Tags are denoted by an `@` prefix followed by the tag name. You can add multiple tags in one command:

```bash
tasq add "Review codebase @work @review"
```

This makes it easy to categorize and filter tasks later.

## Due-Date Syntax

Due dates follow the format `YYYY-MM-DD`. They are optional but highly recommended for prioritizing tasks. For example:

```bash
tasq add "Final exam preparation due 2023-11-15"
```

## Customizing Task File Location

If you prefer to keep your todo list in a different location, set the environment variable `TASQ_FILE` before running tasq:

```bash
export TASQ_FILE=/path/to/your/tasks.txt
tasq add "Buy groceries"
```

This ensures that tasq uses the specified file for all operations.

## Conclusion

tasq is designed to be a straightforward yet powerful tool for managing your tasks without cluttering up complex databases. Its simplicity, combined with support for tags and due dates, makes it an excellent choice for anyone looking to maintain their todo list in plain text files that are version control friendly. Happy tasking!
