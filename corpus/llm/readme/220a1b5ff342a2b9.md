## packlite: Your Lightweight JavaScript & CSS Bundler

Packlite is a simple and fast command-line tool built with Node.js that helps streamline the deployment process for static websites.  Whether you're building a personal blog, portfolio, or small application, packlite bundles your JavaScript and CSS files into a manageable number of output files, minimizing HTTP requests and improving page load times.

**Why packlite?**

Let's face it: complex bundlers like Webpack and Parcel can be overkill for simple static sites. They come with steep learning curves and often bloat build processes unnecessarily. Packlite is designed to be lean and focused, offering just the essential features you need without the fuss.

**Getting Started:**

1. **Installation:**
   ```bash
   npm install -g packlite
   ```

2. **Configuration:** Create a `packlite.config.js` file in your project's root directory. Here's a basic example:

   ```javascript
   module.exports = {
     entryPoints: [
       { 
         name: 'app', 
         path: './src/index.js' 
       },
       {
         name: 'styles',
         path: './src/styles.css'
       }
     ],
     outputDir: 'dist'
   };
   ```

   This config specifies two entry points: `app` (a JavaScript file) and `styles` (a CSS file). They will be bundled into a single output file named "bundle.js" in the `dist` directory. You can customize names, paths, and output directories as needed.

3. **Build:** Run `packlite build` to bundle your files.

4. **Watch Mode:** For local development, use `packlite watch`. This will automatically rebuild your bundles whenever changes are detected in your source files.

5. **Source Maps:** Packlite generates detailed source maps for debugging purposes. Enable them with the `--source-maps` flag: `packlite build --source-maps`.

**Performance Comparison:**

Packlite's minimalist approach shines when comparing build times to larger bundlers. While Webpack and Parcel offer extensive features, they often come with longer compilation times due to their complex configurations and optimizations. Packlite focuses on speed and efficiency, delivering significantly faster builds for simple projects.


**Key Features:**

* **Lightweight & Fast:** Designed for simplicity and speed, perfect for static sites.
* **Easy Configuration:** A straightforward config file gets you started quickly.
* **Watch Mode:** Auto-rebuilds bundles on changes for seamless development.
* **Source Maps:**  Debug your code with ease using generated source maps.

Packlite is the ideal choice if you need a reliable and efficient way to bundle your JavaScript and CSS for deployment. Its simplicity and speed make it a perfect fit for smaller projects where complexity isn't necessary.
