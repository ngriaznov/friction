**phodedupe: Perceptual Hash-based Duplicate Photo Scanner**

Phodedupe is an open-source command-line tool written in Python that scans one or more directories for duplicate photos using perceptual hashing. This approach allows it to identify resized or slightly re-compressed copies of the same image as duplicates.

**Why Perceptual Hashing?**

Traditional methods of detecting duplicates, such as comparing checksums or MD5 hashes, are not effective when dealing with images that have been resampled, compressed, or otherwise altered. Perceptual hashing, on the other hand, creates a hash based on the visual content of an image, making it ideal for identifying similar-looking photos.

**Installing phodedupe**

Phodedupe is available via pip:
```bash
pip install phodedupe
```
**Basic Usage**

To scan a folder and review a report of duplicate groups before deleting anything, run the following command:
```bash
phodedupe -d /path/to/folder
```
This will generate a JSON report detailing duplicate groups. You can then review this report to decide which duplicates to delete.

**Dry-Run Flag**

To avoid accidentally deleting files, use the `--dry-run` flag:
```bash
phodedupe -d /path/to/folder --dry-run
```
This will simulate the deletion process without actually removing any files.

**Configuration Options**

Phodedupe offers several configuration options to customize its behavior:

*   **Hash Algorithm**: Choose from a range of perceptual hashing algorithms, including `phash`, `dhash`, and `whash`.
    ```bash
phodedupe -d /path/to/folder --algorithm whash
```
*   **Similarity Threshold**: Adjust the minimum similarity required for two images to be considered duplicates.
    ```bash
phodedupe -d /path/to/folder --threshold 0.8
```
**Reviewing Reports**

The generated JSON report contains information about each duplicate group, including:

*   Group ID: A unique identifier for each set of duplicates.
*   Image paths: The file paths of the images in each group.
*   Similarity score: A value between 0 and 1 indicating how similar the images are.

To review a report, use a tool like `jq` to parse the JSON output:
```bash
phodedupe -d /path/to/folder | jq '.'
```
This will display the report in a human-readable format. You can then use this information to decide which duplicates to delete.

**Security Considerations**

Phodedupe is designed to be safe and respectful of your files. It only deletes files when instructed to do so, and it preserves the original file paths and metadata. However, as with any tool that manipulates files, use caution when running phodedupe on sensitive data.

By using perceptual hashing, phodedupe provides a robust and accurate method for detecting duplicate photos in your directories. Its configuration options allow you to customize its behavior to suit your specific needs.
