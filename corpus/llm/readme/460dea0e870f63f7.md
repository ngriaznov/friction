# confkeep: Preserving Configuration Format in INI Files
=====================================================

confkeep is a small Python library designed to parse INI-style configuration files while preserving comments and blank lines. This allows users to round-trip their configuration files without losing formatting, making it easier to manage complex configurations.

**Motivation**
---------------

The standard library's `configparser` module does not preserve comments and blank lines in INI files, which can lead to formatting issues when editing the file programmatically. confkeep addresses this limitation by providing a more flexible and user-friendly configuration parsing solution.

**Installation**
-----------------

To install confkeep via pip, run:

```bash
pip install confkeep
```

**Example Usage**
----------------

Here's an example of reading a config file, modifying one value, and writing it back out:
```python
import confkeep

# Read the config file
with open('example.ini', 'r') as f:
    cfg = confkeep.ConfigParser(f)

# Modify the value of the 'database' section
cfg['database']['host'] = 'localhost'

# Write the modified config back to the file
with open('example.ini', 'w') as f:
    cfg.write(f)
```

This code reads an `example.ini` file, modifies the `host` value in the `database` section, and writes the modified config back to the same file.

**Supported Syntax Quirks**
---------------------------

confkeep supports several syntax quirks that are not available in the standard library's `configparser` module:

*   **Inline comments**: confkeep preserves inline comments (`# comment`) in INI files.
*   **Multi-line values**: confkeep allows multi-line values, making it easier to define complex configuration settings.
*   **Section inheritance**: confkeep supports section inheritance, which enables you to inherit settings from parent sections.

**ConfigParser API**
--------------------

confkeep's `ConfigParser` class provides a similar API to the standard library's `configparser` module. You can use it to read and write INI files with ease:

```python
# Read a config file
with open('example.ini', 'r') as f:
    cfg = confkeep.ConfigParser(f)

# Write a new config file
cfg['database'] = {'host': 'localhost'}
with open('new_example.ini', 'w') as f:
    cfg.write(f)
```

**Config File Format**
------------------------

confkeep supports the standard INI file format, with some additional features:

*   **Section inheritance**: confkeep uses a `:` character to separate section names from values.
*   **Multi-line values**: confkeep preserves multi-line values by using a newline (`\n`) character as the delimiter.

**Limitations**
----------------

confkeep is designed to be a lightweight, easy-to-use library. However, it may not offer all the features and flexibility of more advanced configuration parsing solutions. If you need more advanced features or customization options, consider using a different library.

By providing a simple and user-friendly way to parse INI files while preserving comments and blank lines, confkeep makes it easier to manage complex configurations without sacrificing formatting. Try it out today!
