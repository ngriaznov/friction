## tasq - A Simple Command-Line Todo Manager

tasq is a lightweight command-line todo manager designed for simplicity and version control friendliness. It stores your tasks as plain text lines in a file, ensuring your task list remains readable and easily trackable with tools like Git.

**Installation:**

tasq is written in Go and can be installed using `go install`:

```bash
go install github.com/yourusername/tasq@latest
```
(Replace `yourusername` with your GitHub username).  Make sure you have Go installed on your system.

**Usage:**

Basic commands are invoked directly from the command line:

* **Add a task:** `tasq add "My Task Description"`
* **Complete a task:** `tasq complete <task_id>` (Task IDs are sequential numbers assigned automatically)
* **List tasks:** `tasq list` 
* **Help:** `tasq help`


**Data Storage & Configuration:**

By default, tasq stores your tasks in the file `~/.tasq.txt`. You can change this location using the `TASQ_DATA_FILE` environment variable:

```bash
export TASQ_DATA_FILE=/path/to/your/tasks.txt
```

**Task Syntax:**

Tasks are represented as plain text lines.  You can add tags and due dates to tasks for more granular organization:

* **Tags:** Use the `@` prefix to tag tasks (e.g., `tasq add "Buy groceries" @shopping`). Multiple tags can be used, separated by spaces.
* **Due Dates:** Specify a due date in YYYY-MM-DD format (e.g., `tasq add "Prepare presentation" @work 2023-10-27`).

**Filtering Tasks:**

You can filter your task list using tags and due dates:

* **List by Tag:**  `tasq list --tag shopping` (Lists all tasks tagged with 'shopping')
* **List by Due Date:** `tasq list --due 2023-10-27` (Lists all tasks due on 2023-10-27)
* **Combine Filters:** `tasq list --tag shopping --due 2023-10-27` (Lists tasks tagged with 'shopping' and due on 2023-10-27)


**Example Workflow:**

1.  `tasq add "Pay bills" @finance`
2.  `tasq add "Walk the dog" @personal`
3.  `tasq list` (Output: 1: Pay bills @finance, 2: Walk the dog @personal)
4.  `tasq complete 1`
5.  `tasq list` (Output: 2: Walk the dog @personal)

**Version Control Friendly:**

Because tasq stores tasks as plain text files, you can easily track changes and collaborate using Git or other version control systems.


**Further Development:**

This is a basic implementation. Future development could include features like prioritization, recurring tasks, and more sophisticated filtering options.
