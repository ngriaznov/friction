## wavtag: A Lightweight WAV Metadata Library

wavtag is a small C library designed to read and write metadata chunks (LIST/INFO tags) within WAV audio files without modifying the actual audio data. This allows you to easily access and manipulate information like artist, title, album, comment, and more, providing flexibility for applications requiring metadata management in WAV files.

**Key Features:**

* **Non-destructive:** wavtag operates solely on metadata chunks, leaving the audio sample data untouched.
* **Platform-independent:** Designed with portability in mind, handling byte order variations across different systems.
* **Simple API:** Straightforward functions for opening WAV files, reading existing tags, and writing modified tags back to the file.

**Build Instructions:**

1. Ensure you have a working C compiler (e.g., GCC) installed.
2. Create a directory for your project and navigate into it.
3. Clone this repository: `git clone https://github.com/yourusername/wavtag.git`
4. Run the `make` command to compile the library:

```bash
make
```

This will generate a `libwavtag.a` static library file.

**Public API:**

* **`wavtag_open(const char *filename, int mode)`:** Opens a WAV file for reading or writing (`mode`: 0 for read-only, 1 for read-write). Returns a file handle on success, `NULL` otherwise.
* **`wavtag_read_tag(int fh, const char *tag_name)`:** Reads the value of a specific tag from the opened file. Returns a pointer to the tag value string, or `NULL` if the tag is not found.
* **`wavtag_write_tag(int fh, const char *tag_name, const char *value)`:** Writes a new tag or modifies an existing one in the opened file.

**Byte Order Handling:**

wavtag automatically handles byte order conversions to ensure compatibility across platforms (big-endian vs. little-endian). This eliminates the need for manual byte swapping when reading and writing tags.


**Example Usage:**

```c
#include "wavtag.h"

int main() {
  int fh = wavtag_open("audio.wav", 1); // Open for read-write
  if (fh == NULL) {
    fprintf(stderr, "Error opening file\n");
    return 1;
  }

  char *artist = wavtag_read_tag(fh, "ARTIST");
  printf("Artist: %s\n", artist);

  wavtag_write_tag(fh, "COMMENT", "Modified by wavtag");

  wavtag_close(fh); // Close the file
  return 0;
}
```
