# phodedupe - Duplicate Photo Finder
=====================================

phodedupe is an open-source command-line tool that uses perceptual hashing to identify duplicate photos in one or more directories.

**Why Perceptual Hashing?**
---------------------------

Plain checksum comparisons are not sufficient for detecting duplicate photos, as they do not account for changes in image compression, resizing, or other transformations. Perceptual hashing, on the other hand, divides an image into small blocks and generates a unique hash value for each block based on its visual content. This allows phodedupe to detect similar images even if they have undergone these types of transformations.

**Installation**
---------------

To install phodedupe via pip, run the following command:
```bash
pip install phodedupe
```
**Basic Usage**
----------------

Here's an example of how to use phodedupe to scan a folder and review a report of duplicate groups:
```bash
phodedupe --scan /path/to/folder
```
This will create a report of duplicate groups in the specified directory, including the number of duplicates, similarity thresholds, and links to original images.

**Flags**
---------

*   `--dry-run`: Run a dry run of the scanner without actually deleting any files. This is useful for testing purposes or to review the list of duplicate groups before deletion.
*   `--hash-algorithm`: Choose the hashing algorithm to use. Options include `md5`, `sha1`, and `blake2` (default: `blake2`).
*   `--similarity-threshold`: Set the minimum similarity threshold for two images to be considered duplicates (default: 0.8).

**Configuration Options**
-------------------------

You can customize phodedupe's behavior by passing configuration options as command-line arguments or by editing the `phodedupe.conf` file in your home directory.

*   `--config`: Specify a custom configuration file.
*   `--hash-algorithm`: Choose the hashing algorithm to use. (default: `blake2`)
*   `--similarity-threshold`: Set the minimum similarity threshold for two images to be considered duplicates (default: 0.8)
*   `--max-duplicates-per-group`: Limit the number of duplicate groups reported per scan (default: None, all groups included)

**Tips and Tricks**
--------------------

*   Use the `--dry-run` flag when scanning a new folder to review the list of duplicate groups before deletion.
*   Adjust the similarity threshold to suit your needs. A lower value will detect more duplicates but may also catch false positives.
*   You can use phodedupe to scan multiple folders by passing a comma-separated list of directories as an argument.

**Example Configuration File**
-----------------------------

Here's an example `phodedupe.conf` file that sets the hashing algorithm and similarity threshold:
```bash
[DEFAULT]
hash_algorithm = md5
similarity_threshold = 0.9
```
By following these instructions, you can use phodedupe to efficiently find and manage duplicate photos in your image collection.
