# Tasq: A Simple Command-Line Todo Manager

Tasq is a lightweight todo manager designed for simplicity and readability. It stores your tasks as plain text lines in a `.tasq` file, making them easy to read and diff in version control systems like Git.

## Installation

To get started with tasq, first ensure you have Python installed on your system (version 3.x). You can install it using pip:

```bash
pip install tasq
```

Alternatively, if you prefer a more hands-on approach, clone the repository and run `python -m pip install .` in the root directory.

## Basic Usage

### Adding Tasks

To add a new task to your todo list, use the following command:

```bash
tasq add "Finish the report"
```

Tasq will automatically append this task to the `.tasq` file located by default at `~/.tasq` on Linux/Mac or `%APPDATA%\tasq\task.txt` on Windows.

### Completing Tasks

To mark a task as completed, use:

```bash
tasq complete 1
```

This command marks the first task in your list as done. You can specify multiple tasks to complete by providing their indices separated by spaces.

### Listing Tasks

Viewing all tasks is simple with:

```bash
tasq list
```

To filter the list by tags or due dates, see below.

## Tagging and Due Date Syntax

Tasq supports tagging using an `@` prefix before a word. For example:

```bash
tasq add "@work Submit the client proposal"
```

Due date support is also available using the format `2023-10-31`. Add it after your task description, like so:

```bash
tasq add "Finish the presentation 2023-10-31"
```

## Filtering

To filter tasks by tag or due date, use the following commands:

### By Tag

```bash
tasq list @work
```

This will show all tasks tagged with `@work`.

### By Due Date

```bash
tasq list 2023-10-31
```

This command lists tasks that are due on October 31, 2023.

## Task File Location and Environment Variable

By default, tasq looks for the `.tasq` file in your home directory (`~/.tasq` on Linux/Mac or `%APPDATA%\tasq\task.txt` on Windows). However, you can point it to a different location using an environment variable. Set `TASQ_FILE` before running any commands:

```bash
export TASQ_FILE=/path/to/your/tasks.tasq  # For Linux/Mac
set TASQ_FILE=C:\path\to\your\tasks.tasq   # For Windows
```

This allows you to keep your tasks separate from the default location, making it easier to version control or share them.

## Conclusion

Tasq is a straightforward todo manager that keeps your tasks readable and simple. Whether you're managing daily chores or complex projects, tasq's clear text-based approach ensures your todos stay organized and easy to manage.

Happy tasking!
