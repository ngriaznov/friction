# Lumen: A Static Site Generator for Stunning Photography Portfolios

Lumen is a Ruby gem designed to effortlessly create beautiful and responsive static websites tailored for showcasing your photography. With its intuitive folder structure, powerful features, and minimal configuration, Lumen empowers you to focus on what matters most: your stunning visuals. 

## Getting Started

**Installation:**

1. **Install the gem:**
   ```bash
   gem install lumen
   ```

2. **Create a new project:**
   ```bash
   lumen new my-photography-portfolio
   ```

This command generates a basic project structure with essential files and folders. 

**Project Structure:**

Lumen follows a straightforward folder convention:

* **`content/`**: Contains all your Markdown content. Each image gallery should have its own dedicated folder within `content`. For example, `content/portraits/`, `content/landscapes/`, etc.
    * Within each gallery folder, place individual Markdown files for each page (e.g., `gallery-1.md`). These files should only contain the HTML content for the page without any front matter.

* **`images/`**: Holds all your image assets. 

   *  Organize images within subfolders corresponding to their respective galleries in `content`. For example, `images/portraits/IMG_001.jpg`, `images/landscapes/DSC_4567.jpg`.

* **`config/site.yml`**: This YAML file defines your site's configuration:
    * `title`: Your photography portfolio title (e.g., "John Doe Photography").
    * `theme`: The name of the active theme (more details below).
    * `output_dir`: Specifies the directory where Lumen will generate your static website files (default: `public`).

**Themes:**

Lumen offers a selection of pre-built themes to customize the look and feel of your portfolio. You can find available themes in the [themes](https://github.com/yourusername/lumen/tree/main/themes) directory. To use a theme, simply update the `theme` setting in `config/site.yml`.

**Building Your Site:**

To generate your static website:
```bash
cd my-photography-portfolio
lumen build 
```

This command processes your content and images, renders them into HTML files, and places them within the specified `output_dir` (default: `public`).

**Local Preview Server:**

Start a local development server to view your site in real time:

```bash
lumen serve
```

This will open your portfolio in a web browser. Changes you make to your content or theme files will be reflected automatically.

**Key Features:**

* **Thumbnail Generation:** Lumen automatically generates thumbnails for all your images, ensuring optimal loading times and responsiveness.
* **EXIF-Based Caption Extraction:** Leverage EXIF data from your image files to automatically populate captions with relevant information like location, date, and camera model.
* **Clean and Minimal Configuration:** Configure your site title, theme, and output directory in a single YAML file (`config/site.yml`).
