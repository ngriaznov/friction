## Quillfeather Keyboard Shortcuts Reference

Quillfeather is designed for speed and efficiency. This reference guide outlines the available keyboard shortcuts to help you navigate, edit, search, and manage windows with ease.

**Navigation:**

* **Arrow Keys (↑, ↓, ←, →):** Move the cursor one character at a time in the corresponding direction.
* **Ctrl + Arrow Keys:** Move the cursor word by word in the specified direction.
* **Home:** Move to the beginning of the current line.
* **End:** Move to the end of the current line.
* **Page Up/Down:** Scroll up or down one page.
* **Ctrl + Page Up/Down:** Scroll up or down ten lines.

**Editing:**

* **Backspace/Delete:** Delete the character before or after the cursor, respectively.
* **Ctrl + Backspace/Delete:** Delete the entire word before or after the cursor, respectively.
* **Enter:** Insert a new line.
* **Shift + Enter:** Insert a new line and keep the cursor on the same line.
* **Tab:** Indent the current line.
* **Shift + Tab:** Unindent the current line.
* **Ctrl + C/X/V:** Copy, cut, or paste the selected text.
* **Ctrl + Z/Y:** Undo or redo the last action.

**Search:**

* **Ctrl + F:** Open the search dialog box.
* **Enter:** Search for the entered term.
* **Esc:** Close the search dialog box.
* **Shift + Ctrl + F:** Open the find-and-replace dialog box.

**Window Management:**

* **Alt + Tab:** Switch between open windows.
* **Ctrl + N/W:** Create a new window or close the current window, respectively.
* **Ctrl + 1-9:** Switch to the specified numbered window.
* **Ctrl + M:** Minimize all windows except the current one.

**Additional Notes:**

* All shortcuts can be customized within Quillfeather's settings menu.
* Some shortcuts may require specific modifiers depending on your operating system (e.g., Command instead of Ctrl on macOS).


We encourage you to explore and master these shortcuts for a more efficient and enjoyable editing experience in Quillfeather.
