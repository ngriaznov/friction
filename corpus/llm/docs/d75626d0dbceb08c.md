**Getting Started with dedupe-shots: A Command-Line Tool for Removing Duplicate Photos**

Dedupe-shots is a powerful command-line tool that uses perceptual hashing to identify and remove duplicate photos from your local photo library. In this guide, we'll walk you through the installation process, demonstrate how to run a basic scan, and provide instructions on how to preview matches before deletion.

**Installation**

To install dedupe-shots, follow these steps:

1. **Clone the repository**: Run the following command in your terminal:
   ```bash
git clone https://github.com/your-username/dedupe-shots.git
```
2. **Change into the project directory**: Navigate to the cloned repository using the following command:
   ```bash
cd dedupe-shots
```
3. **Install dependencies**: Run the following command to install required dependencies:
   ```bash
npm install
```
4. **Compile and link**: Compile the code by running the following commands:
   ```bash
make build
```

**Basic Scan**

To start scanning your photo library, run the following command:

```bash
./dedupe-shots scan /path/to/your/photo/library
```

Replace `/path/to/your/photo/library` with the actual path to your photo library. The scan will begin by hashing each image in the specified directory and its subdirectories.

**Previewing Matches**

After the scan completes, you'll see a report of duplicate images. To preview matches before deletion, use the following command:

```bash
./dedupe-shots preview /path/to/your/photo/library
```

This will display information about the duplicates found during the scan, including the image paths and similarity scores.

**Deleting Duplicates**

To remove duplicate photos from your library, use the `delete` subcommand. First, you'll need to specify which images to delete using the `-i` flag:

```bash
./dedupe-shots delete -i /path/to/your/photo/library
```

This will list the paths of all duplicate images in the specified directory and its subdirectories.

**Deleting Specific Images**

To delete specific images, use the `-d` flag followed by the image path. For example, to delete an image with a hash of `1234567890abcdef`, run:

```bash
./dedupe-shots delete -d /path/to/your/photo/library/1234567890abcdef.jpg
```

**Example Use Case**

Here's an example of how you can use dedupe-shots in your daily workflow:

1. Install and set up the tool according to the instructions above.
2. Create a backup of your photo library before running the scan or delete subcommand.
3. Run `./dedupe-shots scan /path/to/your/photo/library` to find duplicates.
4. Review the list of duplicate images using `./dedupe-shots preview /path/to/your/photo/library`.
5. Use `./dedupe-shots delete -i /path/to/your/photo/library` or `./dedupe-shots delete -d /path/to/your/photo/library/1234567890abcdef.jpg` to remove duplicate photos.

**Troubleshooting**

If you encounter issues during the scan or deletion process, refer to the `dedupe-shots` documentation for troubleshooting guides and FAQs.

By following this guide, you'll be able to use dedupe-shots to efficiently identify and remove duplicate photos from your local photo library.
