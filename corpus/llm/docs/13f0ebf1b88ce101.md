**Quillfeather Keyboard Shortcuts Reference**

Quillfeather is a powerful terminal-based text editor designed for efficiency and ease of use among command-line enthusiasts. This reference document outlines essential keyboard shortcuts categorized by their functionality: navigation, editing, search, and window management.

### Navigation

- **Ctrl + Arrow Keys**: Move cursor forward or backward by word.
  - `Ctrl + Left`: Move one word back.
  - `Ctrl + Right`: Move one word forward.

- **Home / End**: Jump to the beginning or end of a line.
  - `Home`: Start at the first character of the current line.
  - `End`: Reach the last character of the current line.

- **Page Up / Page Down**: Scroll up or down by full screen.
  - `Page Up`: Move one page up.
  - `Page Down`: Move one page down.

### Editing

- **Ctrl + X**: Cut the selected text to the clipboard.
- **Ctrl + C**: Copy the selected text to the clipboard.
- **Ctrl + V**: Paste the contents of the clipboard.
- **Ctrl + Z**: Undo the last action.
- **Ctrl + Y**: Redo the last undone action.

- **Backspace / Delete**: Remove a character or selection behind or in front of the cursor, respectively.
  - `Backspace`: Delete one character before the cursor.
  - `Delete`: Remove the character under the cursor.

- **Shift + Arrow Keys**: Extend text selection by word or character.
  - `Shift + Left`: Select previous word.
  - `Shift + Right`: Select next word.

### Search

- **Ctrl + F**: Open a search bar to find specific text within the document.
- **Ctrl + R**: Perform a reverse search for previously found text.
- **Ctrl + Shift + K**: Navigate to the next occurrence of the search term.
- **Ctrl + Shift + J**: Jump back to the previous occurrence of the search term.

### Window Management

- **Alt + F4**: Close the current Quillfeather window.
- **Ctrl + W**: Close the active tab/window within Quillfeather.
- **Alt + Tab**: Switch between open Quillfeather windows and other applications.
- **Shift + Ctrl + T**: Open a new terminal tab with Quillfeather.

These shortcuts are designed to streamline your editing experience, allowing for rapid navigation and efficient text manipulation without leaving the keyboard. Familiarizing yourself with these commands can significantly enhance productivity when working within Quillfeather.
