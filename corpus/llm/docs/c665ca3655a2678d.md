##  Declutter Your Photo Library with `dedupe-shots`: A Getting Started Guide

Tired of sifting through hundreds (or thousands!) of near-identical photos? `dedupe-shots` is your command-line companion for efficiently identifying and removing duplicates, freeing up valuable storage space. Leveraging the power of perceptual hashing, it analyzes photo content rather than file metadata, ensuring accurate results even with minor differences.

### Installation: Get Started in Seconds

Installing `dedupe-shots` is a breeze thanks to its simple dependency structure. Open your terminal and execute the following commands:

```bash
pip install dedupe-shots
```

That's it! You're now ready to conquer your photo duplicates.

### Basic Scan: Identify Potential Duplicates

The core functionality of `dedupe-shots` lies in its ability to scan a directory for duplicate images. To initiate a scan, use the following command structure:

```bash
dedupe-shots -p <path_to_photos>
```

Replace `<path_to_photos>` with the actual path to your photo library. 

For example:

```bash
dedupe-shots -p /Users/yourname/Pictures
```

This command will analyze all images within the specified directory and generate a report listing potential duplicate groups, along with their similarity scores.

### Preview Matches: Choose Wisely

Before permanently removing any files, it's crucial to review the identified matches. `dedupe-shots` offers a convenient preview function for this purpose. Execute the following command:

```bash
dedupe-shots -p <path_to_photos> -v
```

This will display thumbnail previews of each potential duplicate group, allowing you to visually assess the similarities and confirm if removal is necessary.

### Removing Duplicates: Make Your Choice

Once you've confidently identified duplicates to remove, utilize the `--remove` flag:

```bash
dedupe-shots -p <path_to_photos> --remove
```

This command will permanently delete all flagged duplicate images from your photo library. Proceed with caution, as this action is irreversible!


### Conclusion

`dedupe-shots` empowers you to efficiently manage your photo library by identifying and removing duplicates with ease. Its simple interface and powerful perceptual hashing algorithm make it a valuable tool for anyone seeking to reclaim storage space and streamline their digital photo collection. Happy decluttering!
