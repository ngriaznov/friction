## Quillfeather Terminal Editor - Keyboard Shortcuts Reference

Quillfeather is a powerful, terminal-based text editor designed for efficient code editing and document manipulation. This document outlines the key keyboard shortcuts available to enhance your workflow.

**I. Navigation:**

These shortcuts help you quickly move around within your documents:

* **`Ctrl + Left Arrow`**: Move cursor one character left.
* **`Ctrl + Right Arrow`**: Move cursor one character right.
* **`Ctrl + Up Arrow`**: Move cursor up one line.
* **`Ctrl + Down Arrow`**: Move cursor down one line.
* **`Alt + Up Arrow`**: Move cursor to the beginning of the current paragraph (if present).
* **`Alt + Down Arrow`**: Move cursor to the end of the current paragraph (if present).
* **`Home`**: Move cursor to the beginning of the document.
* **`End`**: Move cursor to the end of the document.
* **`Page Up`**: Scroll up one page.
* **`Page Down`**: Scroll down one page.
* **`Ctrl + G`**: Go to a specific line number (prompt for line). 
* **`Ctrl + H`**:  Go back one line.
* **`Ctrl + J`**: Go forward one line.


**II. Editing:**

These shortcuts provide quick access to common editing operations:

* **`Ctrl + C`**: Cancel the current operation (e.g., copy/paste, selection).
* **`Ctrl + X`**: Cut selected text or cursor position.  Stores the cut text in the clipboard.
* **`Ctrl + Y`**: Paste previously cut text from the clipboard.
* **`Ctrl + A`**: Select all text in the document.
* **`Ctrl + D`**: Delete character under cursor (works like `Backspace`).
* **`Delete`**: Delete selected text or characters to the right of the cursor.
* **`Ctrl + F`**:  Start searching forward within the document.
* **`Ctrl + B`**: Start searching backward within the document. 
* **`Ctrl + I`**: Indent the current line (useful for code).
* **`Ctrl + U`**: Unindent the current line.
* **`Ctrl + S`**: Save the current file.  (Note: If saved automatically, this may not be necessary.)


**III. Search:**

These shortcuts streamline searching within your documents:

* **`Ctrl + F`**: Start forward search (as described above). Keeps active until cancelled with `Ctrl + C`.
* **`Ctrl + G`**:  Stop forward search and return to normal cursor navigation.
* **`Ctrl + B`**: Start backward search (as described above). Keeps active until cancelled with `Ctrl + C`.
* **`Ctrl + Shift + G`**: Stop backward search and return to normal cursor navigation.

**IV. Window Management:**

These shortcuts manage the Quillfeather window:

* **`Ctrl + T`**:  Toggle between tabs (if multiple files are open).
* **`Ctrl + W`**: Close the current tab or window.
* **`Ctrl + Shift + W`**: Close all tabs and exit Quillfeather.
* **`Ctrl + L`**: Clear the screen – effectively a "clear" command.

**Important Notes:**

*  Quillfeather's keybindings are designed for efficiency within a terminal environment. 
*  Some shortcuts may have slightly different behavior depending on your operating system and terminal configuration. 
*  For more advanced features and customization, consult the Quillfeather documentation (available via the `help` command within the editor).

This reference document provides a starting point for utilizing Quillfeather's keyboard shortcuts effectively. Experiment with these commands to optimize your editing workflow!
