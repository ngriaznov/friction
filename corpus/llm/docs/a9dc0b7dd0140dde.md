### Reference Document for Keyboard Shortcuts in Quillfeather (Terminal-Based Text Editor)

Quillfeather is a powerful terminal-based text editor designed to enhance productivity through efficient keyboard shortcuts. This document categorizes the available shortcuts into four main groups: Navigation, Editing, Search, and Window Management.

---

#### 1. **Navigation**

- **Home (`Ctrl+A`)**: Move cursor to the beginning of the line.
- **End (`Ctrl+E`)**: Move cursor to the end of the line.
- **Left Arrow (`Ctrl+B`)**: Move left one character.
- **Right Arrow (`Ctrl+F`)**: Move right one character.
- **Up Arrow (`Ctrl+P`)**: Move up one line in history.
- **Down Arrow (`Ctrl+N`)**: Move down one line in history.
- **Page Up (`Ctrl+D` or `Shift+PgUp`)**: Scroll up one page.
- **Page Down (`Ctrl+U` or `Shift+PgDn`)**: Scroll down one page.

#### 2. **Editing**

- **Delete (`Ctrl+D`)**: Delete the character under the cursor.
- **Backspace (`Ctrl+H` or `Alt+B`)**: Delete the character to the left of the cursor.
- **Insert (`Ctrl+I` or `Ctrl+O`)**: Toggle insert mode (default is overwrite).
- **Copy (`Ctrl+K`)**: Copy selected text to clipboard.
- **Paste (`Ctrl+Y`)**: Paste text from clipboard at current cursor position.
- **Cut (`Ctrl+X`)**: Cut the selected text to clipboard.
- **Redo Last Action (`Ctrl+R` or `Alt+Shift+Y`)**: Redoes the last action undone with `Ctrl+U`.

#### 3. **Search**

- **Find (`Ctrl+F` or `Alt+O`)**: Open a dialog to search for text within the document.
- **Replace (`Ctrl+R` or `Alt+X`)**: Replace text found during the find operation.
- **Next Match (`F8` or `Ctrl+G`)**: Move to the next match of the current search term.
- **Previous Match (`Shift+F8` or `Ctrl+Shift+G`)**: Move to the previous match of the current search term.

#### 4. **Window Management**

- **Split Vertically (`Ctrl+V` or `Alt+S`)**: Split the current window into two vertically, with each half being a new editor.
- **Split Horizontally (`Ctrl+H` or `Alt+D`)**: Split the current window into two horizontally, with each half being a new editor.
- **Close Window (`Ctrl+W` or `Alt+C`)**: Close the currently active split window.
- **Toggle Fullscreen (`F12`)**: Toggle full-screen mode on and off to maximize or restore the terminal size.

---

**Additional Tips:**

- **Undo/Redo History**: Use `Ctrl+U` (undo) and `Ctrl+R` (redo) to navigate through your editing history.
- **Marking Text**: You can mark text by dragging over it with your cursor. This allows you to select multiple lines or a range of text for operations like copy, cut, and paste.

This document provides an overview of the essential shortcuts in Quillfeather, aimed at enhancing user efficiency through keyboard-driven operations. For more detailed instructions on advanced features or troubleshooting tips, refer to the official Quillfeather documentation or support resources.
