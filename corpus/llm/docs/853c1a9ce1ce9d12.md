**Building a Simple Webhook Receiver in Node.js with Hookline**
===========================================================

In this tutorial, we will walk through building a simple webhook receiver using the Hookline middleware library in Node.js. We'll cover installing the package, setting up a basic middleware chain, verifying an incoming signature, and responding with a 200 status.

**Step 1: Install Hookline**
---------------------------

To get started, install Hookline using npm or yarn:

```bash
npm install hookline
# or
yarn add hookline
```

**Step 2: Create a New Node.js Project**
--------------------------------------

Create a new directory for your project and initialize it with `npm init`. Then, create a new file called `app.js` to serve as the main entry point for our application.

```bash
mkdir webhook-receiver
cd webhook-receiver
npm init -y
touch app.js
```

**Step 3: Set Up Hookline Middleware**
--------------------------------------

In your `app.js`, require Hookline and set up a basic middleware chain that will verify incoming requests:

```javascript
// app.js

const express = require('express');
const hookline = require('hookline');

const app = express();

app.use(hookline());
```

**Step 4: Define the Webhook Endpoint**
--------------------------------------

Create a new file called `webhook-endpoint.js` to define our webhook endpoint:

```javascript
// webhook-endpoint.js

const express = require('express');
const { verify } = require('hookline').middleware;

const app = express();

app.post('/webhook', verify({
  secret: 'YOUR_WEBHOOK_SECRET',
  method: ['POST'],
}), (req, res) => {
  // Handle incoming request data
});

module.exports = app;
```

Replace `YOUR_WEBHOOK_SECRET` with a secure secret key that will be used to sign and verify incoming requests.

**Step 5: Verify the Incoming Signature**
--------------------------------------

In your `webhook-endpoint.js`, use the `verify` middleware function to check if the incoming request's signature matches the expected value. If it does, call the next handler in the chain:

```javascript
// webhook-endpoint.js

const express = require('express');
const { verify } = require('hookline').middleware;

const app = express();

app.post('/webhook', verify({
  secret: 'YOUR_WEBHOOK_SECRET',
  method: ['POST'],
}), (req, res) => {
  const signature = req.headers['x-hub-signature'];

  if (!signature || !verify(req.body, signature)) {
    return res.status(401).send('Invalid signature');
  }

  // Handle incoming request data
}, (err) => {
  console.error(err);
});

module.exports = app;
```

**Step 6: Respond with a 200 Status**
--------------------------------------

Finally, in your `webhook-endpoint.js`, respond to the incoming request with a 200 status code:

```javascript
// webhook-endpoint.js

const express = require('express');
const { verify } = require('hookline').middleware;

const app = express();

app.post('/webhook', verify({
  secret: 'YOUR_WEBHOOK_SECRET',
  method: ['POST'],
}), (req, res) => {
  const signature = req.headers['x-hub-signature'];

  if (!signature || !verify(req.body, signature)) {
    return res.status(401).send('Invalid signature');
  }

  console.log(`Received request data: ${JSON.stringify(req.body)}`);
  res.status(200).send('Request received successfully');
}, (err) => {
  console.error(err);
});

module.exports = app;
```

**Step 7: Start the Application**
---------------------------------

Start your application using `node`:

```bash
node app.js
```

Your webhook receiver is now live and ready to receive incoming requests!

That's it! You've successfully built a simple webhook receiver in Node.js using Hookline. This tutorial covered installing the package, setting up a basic middleware chain, verifying an incoming signature, and responding with a 200 status.
