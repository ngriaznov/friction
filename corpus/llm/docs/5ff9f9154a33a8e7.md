Quillfeather Terminal-Based Text Editor Reference Document

**Navigation**

1. **Move cursor**: `Ctrl + A` - Move cursor to the beginning of the line.
2. **Move cursor to end of line**: `Ctrl + E` - Move cursor to the end of the line.
3. **Jump to next/previous line**: `Ctrl + F` / `Ctrl + B` - Jump to the next/previous line in the file.
4. **Scroll up/down**: `Ctrl + R` / `Ctrl + S` - Scroll up/down one screen at a time.
5. **Go to beginning/end of file**: `Ctrl + Home` / `Ctrl + End` - Go to the beginning/end of the file.

**Editing**

1. **Cut text**: `Ctrl + X` - Cut selected text.
2. **Copy text**: `Ctrl + C` - Copy selected text.
3. **Paste text**: `Ctrl + V` - Paste copied text.
4. **Delete character**: `Ctrl + Delete` - Delete the character under the cursor.
5. **Undo/redo**: `Ctrl + Z` / `Ctrl + Y` - Undo/redo last action.

**Search**

1. **Find next occurrence**: `Ctrl + F` - Find next occurrence of search term.
2. **Find previous occurrence**: `Ctrl + Shift + F` - Find previous occurrence of search term.
3. **Replace text**: `Ctrl + H` - Replace selected text with new text.
4. **Search for word**: `Ctrl + S` - Search for the entire word under the cursor.

**Window Management**

1. **Split window horizontally**: `Ctrl + Shift + >` - Split window horizontally.
2. **Split window vertically**: `Ctrl + Shift + <` - Split window vertically.
3. **Merge windows**: `Ctrl + Shift + M` - Merge two windows into one.
4. **Close current window**: `Ctrl + W` - Close the current window.
5. **Switch between windows**: `Ctrl + Tab` / `Ctrl + Shift + Tab` - Switch between open windows.

**Additional Shortcuts**

1. **Toggle full-screen mode**: `F11` or `Fn + F11` - Toggle full-screen mode on/off.
2. **Increase/decrease font size**: `Ctrl + Plus sign` / `Ctrl + Minus sign` - Increase/decrease font size.
3. **Save file**: `Ctrl + S` - Save the current file.

**Customization**

Quillfeather allows for customization of keyboard shortcuts through the Quillfeather settings menu. Users can create their own custom shortcuts by assigning a key combination to an existing action or creating a new action with a unique key combination.

Note: The above list only includes the most commonly used keyboard shortcuts in Quillfeather and may not be exhaustive. For more information on available shortcuts, please refer to the Quillfeather user manual or contact our support team.
