**Getting Started with dedupe-shots: Remove Duplicate Photos from Your Local Library**

Are you tired of sifting through your massive photo collection, only to find multiple copies of the same image? `dedupe-shots` is a command-line tool designed to help you efficiently identify and remove duplicate photos using perceptual hashing. In this guide, we'll walk you through installation, basic usage, and how to preview matches before deleting duplicates.

**Installation**

To install `dedupe-shots`, follow these steps:

1. **Check system requirements**: Ensure your machine meets the minimum requirements:
	* Python 3.6+
	* NumPy
	* Pillow (PIL)
2. **Clone the repository**: Run the following command in your terminal: `git clone https://github.com/your-username/dedupe-shots.git`
3. **Install dependencies**: Navigate to the cloned directory and run `pip install -r requirements.txt` to install NumPy and Pillow.
4. **Run setup script**: Execute `python setup.py install` to finalize the installation.

**Basic Scan Command**

To start scanning your photo library, use the following command:
```bash
dedupe-shots scan /path/to/your/photo/library
```
Replace `/path/to/your/photo/library` with the actual path to your photo collection. This command will create a `scan` directory in the root of your library and begin processing images.

**Understanding Scan Output**

After running the scan command, you'll find several output files:

* **`matches.csv`**: A CSV file containing duplicate matches, including image paths and similarity scores.
* **`hashes.json`**: A JSON file storing perceptual hashes for each processed image.

**Previewing Matches**

Before deleting any duplicates, it's essential to review the `matches.csv` file. You can use a spreadsheet editor like Google Sheets or Microsoft Excel to analyze the data. Sort the file by similarity score ( descending order) and examine the images listed in each duplicate group.

**Deleting Duplicates**

Once you've reviewed the matches, run:
```bash
dedupe-shots delete /path/to/your/photo/library
```
This command will remove duplicate images from your library based on the `matches.csv` file. Be cautious when deleting files; make sure to review the output carefully before proceeding.

**Tips and Variations**

* **Customizing scan settings**: You can modify the `scan` command by adding options:
	+ `-s` or `--size-threshold`: Specify a minimum image size threshold (default: 100KB).
	+ `-q` or `--quality-threshold`: Set a minimum image quality threshold (default: 80%).
* **Ignoring certain directories**: Use the `--ignore` option to exclude specific folders from scanning:
```bash
dedupe-shots scan /path/to/your/photo/library --ignore '/path/to/exclude/folder'
```
By following this guide, you should be able to efficiently remove duplicate photos from your local library using `dedupe-shots`. Remember to always review matches before deleting any files.
