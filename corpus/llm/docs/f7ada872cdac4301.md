# Getting Started Guide for `dedupe-shots`

## Introduction

`dedupe-shots` is a powerful command-line tool designed to help you find and remove duplicate photos across your local photo library using perceptual hashing. This guide will walk you through the installation process, how to perform a basic scan of your photo collection, and how to preview matches before any deletion occurs.

## Installation

To get started with `dedupe-shots`, you need to install it on your system. Follow these steps:

1. **Ensure Python is Installed**: `dedupe-shots` requires Python 3.x. You can check if Python is installed by running `python --version` in your terminal. If not, download and install it from [python.org](https://www.python.org).

2. **Install the Tool**: Use pip to install `dedupe-shots`. Open a terminal and run:
   ```
   pip install dedupe-shots
   ```

3. **Verify Installation**: After installation, verify that `dedupe-shots` is correctly installed by running:
   ```
   dedupe-shots --version
   ```
   You should see the version number printed in the console.

## Basic Scan Command

Once installed, you can start using `dedupe-shots` to scan your photo library. The basic command structure is as follows:

```
dedupe-shots [options] <path_to_photo_library>
```

### Example Command

To scan a directory named `Photos` located in your home folder, use:
```
dedupe-shots ~/Photos
```

### Options Explained

- **`--hash-type=<type>`**: Specify the hashing algorithm to use (e.g., `ssdeep`, `sha256`). The default is `ssdeep`.
- **`--threshold=<value>`**: Set the similarity threshold for considering two images as duplicates. Values range from 0 to 1, where a higher value means stricter matching.
- **`--preview`**: Enable preview mode to display matches before deletion.

## Previewing Matches

Before deleting any files, it's crucial to review potential duplicates. Use the `--preview` option to see which images are considered duplicates:

```
dedupe-shots --preview ~/Photos
```

This command will output a list of duplicate groups found in your photo library, along with their file paths and similarity scores.

### Example Output

```
Duplicate Group: 1
- /home/user/Photos/image1.jpg (similarity: 0.92)
- /home/user/Photos/image2.jpg (similarity: 0.91)

Duplicate Group: 2
- /home/user/Photos/image3.jpg (similarity: 0.95)
- /home/user/Photos/image4.jpg (similarity: 0.94)
```

### Deleting Duplicates

After reviewing the preview, you can proceed to delete duplicates by running:

```
dedupe-shots --delete ~/Photos
```

This command will remove all files marked as duplicates based on your specified threshold and hashing algorithm.

## Advanced Usage

For more advanced users, `dedupe-shots` supports additional features such as customizing the output format for previews or integrating with other tools via scripting. Refer to the official documentation for further details on these capabilities.

## Conclusion

`dedupe-shots` simplifies managing large photo collections by automating duplicate detection and removal using perceptual hashing. By following this guide, you can efficiently organize your photos while ensuring that only truly unique images remain in your library. Happy organizing!
